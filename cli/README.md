# CLI template

TypeScript template for a Node.js CLI library.
JavaScript build is bundled with Rollup, and `.d.ts` files are generated by `tsc`.

## Install deps

```bash
npm install
```

## Build

```bash
npm run build
```

## Run local command

```bash
node dist/cli.js --name Max
```

## Config parsing

CLI automatically looks for one of these files in current directory:

- `infinityloop.config.js`
- `infinityloop.config.mjs`
- `infinityloop.config.cjs`

Example `infinityloop.config.js`:

```js
module.exports = {
  name: "From config"
};
```

Run with auto-detected config:

```bash
node dist/cli.js
```

Run with explicit config path:

```bash
node dist/cli.js --config ./infinityloop.config.mjs
```

## Help

```bash
node dist/cli.js --help
```
